In windows you have to install minikube cluster and docker desktop

Docker Desktop is a open source software. Install from any browser to your system.

Minikube install guide -> 

    choco install minikube -y

    minikube version

    change the context -> docker context use desktop-linux

    install kubectl -> choco install kubernetes-cli -y

    to get the kubectl version -> kubectl version --client

    to start minikube -> minikube start --driver=docker

    Now we have add some addons to minikube

        minikube addons enable metrics-server  # to see the usuage of different services inside the cluster
        minikube addons enable dashboard # to add dashboard
        minikube addons enable ingress # to add ingress create a link between frontend and backend
        minikube addons list # to see the addons are added
        minikube dashboard # to check the dashboard

Required Images are pushed to Docker Hub so we just need to apply the manifest files


Apply the manifest files sequentially

    kubectl apply -f K8s-manifest-file/namespace.yaml
    kubectl apply -f K8s-manifest-file/secret.yaml
    kubectl apply -f K8s-manifest-file/configmap.yaml

    kubectl apply -f K8s-manifest-file/sqlserver/
    kubectl apply -f K8s-manifest-file/db-init/job.yaml

    kubectl apply -f K8s-manifest-file/backend/
    kubectl apply -f K8s-manifest-file/frontend/

    kubectl apply -f K8s-manifest-file/ingress.yaml

Go to this path of your local sytem and open the "host" file in admin mode

    C:\Windows\System32\drivers\etc

    add this -> 127.0.0.1 usermanagement.local



Open a terminal and run this command 

    minikube tunnel
    now to broswer and hit -> usermanagement.local
